## Introduction

The **FoBE Breakout CMSIS-DAP Lite** integrates the [WCH CH32V203F6P6](https://atta.szlcsc.com/upload/public/pdf/source/20230523/1B381532B53560847D0152FAE396CF1C.pdf) microcontroller running CMSIS-DAP firmware, providing a professional debugging and programming solution for ARM Cortex-M based microcontrollers. The module supports JTAG, SWD debugging interfaces, virtual serial port communication, and offers both 3.3V and 5V power output capabilities for target board supply.

## Applications

- ARM Cortex-M microcontroller debugging and programming
- STM32, nRF51/52, and other Cortex-M development
- Educational and prototyping projects
- Firmware development and testing
- Hardware-in-the-loop testing systems

## Key Features

**CMSIS-DAP Interface**
    - Compatible with all ARM Cortex-M based microcontrollers
    - Support for STM32, nRF51/52, and other Cortex-M series
    - JTAG and SWD debugging interfaces
    - Virtual serial port communication
    - Drag-and-drop programming support
    - STLINK compatible and upgradeable firmware

**Development Environment Support**
    - Keil/MDK development environment
    - IAR Embedded Workbench
    - OpenOCD open-source debugging tool
    - ARM Mbed online compiler

**Power Supply**
    - USB-C power input: 5V
    - Dual voltage output: 3.3V and 5V
    - Direct target board power supply capability
    - Power output protection circuits

**Connectivity**
    - 12-Pin 2.54mm Header Interface
    - USB-C connector for host communication
    - Status LED indicators
