## Introduction

CMSIS DAP/DAPLink Emulator JTAG SWD serial Port/ U Disk Drag and Drop Super /STLINK Upgradeable Firmware Emulator. Meet the daily development, debugging and download needs of electronic engineers.


## Key Features

- Target chip: All chips based on Cortex-M core, such as STM32, NRF51/52, etc.
- Debug interface: JTAG, SWD, virtual serial port
- Development environment: Keil/MDK, IAR, OpenOCD
- Input voltage: 5V (USB power supply)
- Output voltage: 5V/3.3V (can directly supply power to the target board)
- 12Pin 2.54mm Header Holes 
- USB-C cable x 1
