## Introduction

The FoBE Breakout L76K is a compact GNSS/GPS module designed for location tracking and navigation applications. It features the L76K GPS receiver IC and offers flexible connectivity options for easy integration into various projects.

This board provides both a SH1.0 6-pin connector and reserved 6-pin header holes for external connections, offering versatility in interfacing with other devices. It also includes a 3.1V rechargeable button cell battery for data retention, an IPEX antenna connector for external GPS antenna, and a PPS indicator LED for indicating GPS pulse-per-second output.

## Key Features

*   **GNSS/GPS Module:**
    *   L76K GPS Receiver IC
    *   Supports GPS, GLONASS, BeiDou, Galileo, and QZSS
    *   PPS Indicator LED

*   **Connectivity:**
    *   SH1.0 6Pin Connector and Reserved 6Pin Header Holes
    *   IPEX Antenna Connector

*   Power supply
    *   3.3V Operating Voltage
    *   3.1V Rechargeable Button Cell Battery for Data Retention
