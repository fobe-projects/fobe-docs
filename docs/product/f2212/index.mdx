## Introduction

The **FoBE Breakout MFP-HUB** is an interface converter that splits a 6-pin MFP (Multi-Function Port) into two 4-pin I¬≤C interfaces. This allows for connecting multiple I¬≤C devices to a single MFP port, simplifying wiring and expanding connectivity options.

## Key Features

- **Interface Conversion**: Converts one 6-pin MFP interface to two 4-pin I¬≤C interfaces.
- **Flexible Connectivity**: Provides one 6-pin JST SH1.0 connector for MFP input and two 4-pin JST SH1.0 connectors for I¬≤C output.
- **On-board Pull-ups**: Each I¬≤C port is equipped with pull-up resistors, which can be individually disconnected via jumpers.
- **Power Indicator**: An on-board power LED indicates the board's status, which can be disabled for low-power applications.
