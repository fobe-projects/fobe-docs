## Introduction

FoBE Breakout AS5600 is a programmable 12-bit high-resolution contactless magnetic rotary position sensor. The FoBE Breakout AS5600 can work as a magnetic potentiometer or a magnetic encoder with excellent reliability and durability.

FoBE Breakout AS5600 has advantages: high precision, non-contact, no rotation angle limitation. All those advantages making it suitable for non-contact angle measurement applications, such as the robot arm, tripod head, motor closed-loop control, machine tool axis positioning.

## Applications
- Motor closed-loop control
- Robot arm
- Tripod head
- Industrial machine control

## Key Features

*   **AS5600:**
    - Highest reliability and durability : Contactless angle measurement
    - Simple programming: Simple user-programmable start and stop positions over the I²C interface
    - Great flexibility on angular excursion : Maximum angle programmable from 18° up to 360°
    - High-resolution output signal : 12-bit DAC output resolution
    - Selectable output : Analog output ratiometric to VDD or PWM-encoded digital output
    - Low-power consumption : Automatic entry into low-power mode
    - Easy setup : Automatic magnet detection
    - Robust environmental tolerance :Wide temperature range: -40°C to 125°C

*   **Connectivity:**
    -  Two 4Pin SH1.0 Connectors(IIC)
    -  5Pin 2.54mm Header Holes (IIC + interrupt)
