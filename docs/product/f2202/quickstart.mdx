## Mechanical dimensions

FoBE Breakout SHT40 is a single-sided 25.4mm x 25.4mm (1" x 1") 1.6mm thick PCB with two SH1.0 4-pin connectors and a set of 4-pin 2.54mm header holes. Fixing by 4 x 1.6mm Screw holes.

## Interfaces

The board provides two 4-pin SH1.0 connectors and a set of 4-pin 2.54mm header holes, all providing the same IIC connections for MCU control.

  |Interface                           |Description |
  |-------------------------------|---------|
  |3V3 |Power supply       |
  |GND  |Ground       |
  |SDA |I2C bus data line       |
  |SCL |I2C bus clock line       |

## Configuration Jumper

The board features a jumper pad:

|Interface                           |Description |
|-------------------------------|---------|
|LED-JUMPER |The PPS (Pulse Per Second) LED jumper. To save power, you can physically cut the trace between the pads of this jumper. You can later re-solder the jumper to reconnect the PPS LED. |
