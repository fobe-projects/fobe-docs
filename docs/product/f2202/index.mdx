## Introduction

The FoBE Breakout SHT40 is a compact digital temperature and humidity sensor module designed for environmental monitoring and data logging applications. It features the SHT40 sensor IC and offers flexible connectivity options for easy integration into various projects.

This board provides two 4-pin SH1.0 connectors and a set of 4-pin 2.54mm header holes, all providing the same IIC connections for MCU control. It also includes a jumper that can be used to turn off the power indicator LED.

## Key Features

*   **Sensor Module:**
    *   SHT40 Temperature and Humidity Sensor IC
    *   Digital IIC interface

*   **Connectivity:**
    *   Two 4Pin SH1.0 Connectors and Reserved 4Pin Header Holes (IIC)

*   **Other:**
    *   Power Indicator LED with Jumper
