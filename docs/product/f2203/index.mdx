## Introduction
FoBE Breakout OLED 0.42 inch is 0.42 inch 72x40 OLED display module.

It has a 0.42 inch 72x40 pixels Graphic Monochrome  olede (SSD1315).

Provides two 4-pin SH1.0 connectors and a set of 4-pin 2.54mm header holes, all providing the same IIC connections for MCU control.

## Applications
- Electronic Products
- Medical apparatus and equipment
- Traffic Safety
- Wearable devices


## Key Features
*   **OLED Display:**
