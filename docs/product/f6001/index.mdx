## Introduction

The FoBE Core nRF52840 Mesh is a powerful and versatile LoRa/LoRaWAN module designed for a wide range of IoT applications. It combines the Nordic Semiconductor nRF52840 microcontroller with the Semtech SX1262 LoRa transceiver, offering excellent performance, long-range communication, and ultra-low power consumption. All pins are brought out as 90 pads (including GND).

## Applications

*   **IoT Devices:** Ideal for creating various IoT devices with LoRa/LoRaWAN connectivity.
*   **Smart Agriculture:** Deploy in agricultural settings for sensor data transmission and remote control.
*   **Smart Cities:** Enable smart city applications such as environmental monitoring and smart street lighting.
*   **Asset Tracking:** Track valuable assets with long-range LoRa communication.
*   **Custom Wireless Solutions:** Build custom wireless solutions with flexible I/O and communication protocols.

## Key Features

*   **Microcontroller:**
    *   Nordic Semiconductor nRF52840
    *   ARM Cortex-M4F processor
    *   64 MHz clock frequency
    *   1 MB Flash, 256 KB RAM

*   **LoRa Transceiver:**
    *   Semtech SX1262
    *   LoRa/LoRaWAN protocol support
    *   Frequency bands: 863-870 MHz (EU868), 902-928 MHz (US915), and others
    *   Maximum output power: +22 dBm

*   **Connectivity:**
    *   All nRF52840 pins available as 90 pads (including GND) for flexible connectivity.
    *   Onboard IPEX antenna connector for LoRa
    *   Onboard IPEX antenna connector for Bluetooth 

*   **Other:**
    *   Ultra-low power consumption for extended battery life
