This site is built with Docusaurus, hosted on GitHub Pages, and may be served via Cloudflare CDN.

We respect your privacy. Below is our privacy policy regarding data collection and usage:

## 1. No Personal Data Collection

This is a static documentation website with no user registration, login, or interactive features. We do not actively collect your name, email, or other personally identifiable information.

## 2. Server Access Logs

Our hosting server may log standard access information, including but not limited to:

- IP address
- Browser type
- Operating system
- Visit time and page path

This data is used solely for website maintenance and traffic analysis.

## 3. Third-Party Services

We use the following third-party services which may collect anonymized data:

- **Google Analytics**: For traffic analysis. See their privacy policy: https://policies.google.com/privacy  
- **Cloudflare CDN** (if enabled): For content delivery optimization. Privacy policy: https://www.cloudflare.com/privacypolicy/

These services may use cookies or similar technologies for anonymous tracking.

If you have any questions about this Privacy Policy, please contact us at: [privacy@fobestudio.com](mailto:privacy@fobestudio.com).
